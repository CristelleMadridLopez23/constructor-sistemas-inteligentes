Guía del Sistema de Gestión de Preguntas

Bienvenido al Constructor de Sistemas Inteligentes. Aquí encontrarás la guía para crear, editar y organizar preguntas que serán la base del motor de inferencia.


1. ¿Qué es este sistema?

Es una herramienta para ingenieros de conocimiento que permite diseñar preguntas y reglas lógicas que servirán como base para un motor de inferencia (similar a un sistema experto).

2. ¿Cómo crear una pregunta?

Ve al menú ❓ Questions, escribe la pregunta, selecciona si es respuesta simple o múltiple, y agrega hasta 8 opciones de respuesta.

3. ¿Cómo definir importancia?

Cada pregunta puede clasificarse como Baja, Media o Alta. Esto ayuda a priorizar su impacto en el sistema.

4. ¿Cómo guardar y cargar preguntas?

Usa el botón 💾 Save Question para exportar tus preguntas a un archivo, y 📂 Load Question para cargarlas en otra sesión.

5. ¿Qué pasa con las preguntas relacionadas?

Puedes configurar condiciones para que una pregunta solo aparezca si otra fue respondida de cierta manera. Ejemplo: Si el usuario responde "Sí" a fiebre, preguntar temperatura exacta.

❓ Preguntas Frecuentes

  • ¿Puedo tener más de 8 opciones de respuesta? → No, el límite es 8.
  • ¿Qué pasa si no relaciono una pregunta? → Aparecerá como pregunta independiente.
  • ¿Dónde se guardan los archivos? → Se descargan en tu dispositivo en formato JSON o similar.
